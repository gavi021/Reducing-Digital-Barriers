Personal Statement

This website was designed with accessibility as a priority. It aims to demonstrate inclusive web design practices informed by the Web Content Accessibility Guidelines (WCAG).

Accessibility Features

  • Semantic HTML
  • Keyboard navigation
  • Readable typography
  • High color contrast

Limitations and Feedback

Accessibility is an ongoing process, and no website is perfect. If you encounter any accessibility barriers while using this site, feedback is welcome and encouraged.
